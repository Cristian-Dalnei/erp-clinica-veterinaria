ERP Clínica Veterinária

Sistema desenvolvido em Python para gerenciamento de uma clínica veterinária, permitindo o cadastro e gerenciamento de clientes e animais, além do controle de acesso baseado no perfil do usuário.

Funcionalidades
Cadastro de clientes
Pesquisa de clientes
Atualização de dados cadastrais
Cadastro de animais
Relacionamento entre clientes e seus animais
Controle de acesso por perfil de usuário
Exibição de menus conforme as permissões do usuário
Tecnologias Utilizadas
Python
PostgreSQL
Psycopg
Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de programação, modelagem de banco de dados e regras de negócio, simulando um sistema utilizado em clínicas veterinárias.

Estrutura do Sistema

O sistema realiza o gerenciamento de clientes e seus respectivos animais, armazenando as informações em banco de dados PostgreSQL. Também possui controle de permissões para diferentes tipos de usuários, exibindo menus específicos conforme o perfil de acesso.
